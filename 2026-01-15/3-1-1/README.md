# 실습 과제: SaaS 서비스 비교 분석하기

## 목표

- 평소 사용하는 서비스가 **SaaS인지 분류**하고,  
  **IaaS / PaaS / SaaS의 차이**를 감으로 이해한다.

---

## 1 - 평소 사용하는 서비스 5개 목록

1. YouTube
2. Netflix
3. KakaoTalk
4. Naver Mail
5. Notion

---

## 2 - 각 서비스가 SaaS인지 분류하기

| 서비스     | 분류 | 이유(한 줄)                                                                   |
| ---------- | ---- | ----------------------------------------------------------------------------- |
| YouTube    | SaaS | 웹/앱에서 바로 사용하며, 설치 없이 영상 플랫폼 기능을 제공하기 때문           |
| Netflix    | SaaS | 웹/앱에서 바로 사용하며, 구독 기반으로 콘텐츠 스트리밍 서비스를 제공하기 때문 |
| KakaoTalk  | SaaS | 클라우드 기반으로 사용자에게 소프트웨어를 제공하는 방식으로 운영되기 때문     |
| Naver Mail | SaaS | 이메일 기능을 웹에서 제공하며 사용자는 계정만 있으면 되기 때문                |
| Notion     | SaaS | 노트·데이터베이스 기능을 웹 기반으로 제공하기 때문                            |

---

## 3 - SaaS가 아닌 경우라면?

위 5개는 모두 SaaS라서 SaaS가 아닌 서비스가 없음

---

## 4 - Netflix의 SaaS 특징 3가지 작성하기

### 1. 웹/앱 접속 기반 서비스

- 사용자는 프로그램을 개발하거나 서버를 설치하지 않고, 로그인 후 바로 서비스 이용이 가능하다.

### 2. 구독 요금제

- 서비스는 월 정액 구독 모델로 운영되며, 사용자는 이용 권한을 결제해서 사용하는 구조다.

### 3. 자동 업데이트와와 운영은 제공자가 담당

- 앱 기능 개선, 콘텐츠 추천 알고리즘, 스트리밍 안정화 등은 Netflix가 서버에서 자동으로 관리/업데이트한다.
- 사용자는 별도로 설치/패치 관리가 필요 없다.
