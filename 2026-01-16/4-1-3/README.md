# 실습 과제: 상대 코드별 응답 시뮬레이션하기

## 목표

**httpsta.us** 서비스를 활용하여 다양한 HTTP 상태 코드를 직접 확인하고,  
각 코드의 의미를 이해한다.

---

## httpsta.us/404와 httpsta.us/500 에러 코드

httpsta.us/404 - 404 Not Found  
httpsta.us/500 - 500 Internal Server Error

## 4xx(클라이언트 에러)와 5xx(서버 에러)의 차이점

4xx - 클라이언트에 문제가 있음을 나타냄
5xx - 서버에 문제가 있음을 나타냄
